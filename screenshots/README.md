This directory contains screenshots of the SQL Injection Detection System POC.

## Required Screenshots

Please capture and save the following screenshots from http://localhost:3000:

1. **dashboard.png** - Dashboard view showing statistics and recent activity
2. **query-tester-benign.png** - Query Tester with a benign query result
3. **query-tester-malicious.png** - Query Tester with a malicious query result

## How to Capture

### Using Browser Screenshot Tools

**Chrome/Edge:**
1. Open Developer Tools (F12)
2. Press `Ctrl+Shift+P` (Windows) or `Cmd+Shift+P` (Mac)
3. Type "screenshot" and select "Capture full size screenshot"

**Firefox:**
1. Press `Ctrl+Shift+S` (Windows) or `Cmd+Shift+S` (Mac)
2. Select the area or choose "Save full page"

**Safari:**
1. Enable Developer menu: Preferences > Advanced > Show Develop menu
2. Develop > Show Web Inspector
3. Right-click on page > Inspect Element > Take Screenshot

### Using System Tools

**macOS:**
- `Cmd + Shift + 4` - Select area to capture
- `Cmd + Shift + 3` - Capture entire screen

**Windows:**
- `Win + Shift + S` - Snipping tool
- `PrtScn` - Capture screen

**Linux:**
- `PrtScn` or use Screenshot application

## Screenshot Specifications

- **Format:** PNG (preferred) or JPG
- **Resolution:** At least 1920x1080 for full page
- **Quality:** High quality, clear text
- **Browser:** Any modern browser (Chrome, Firefox, Safari, Edge)
- **Zoom Level:** 100% (default)
